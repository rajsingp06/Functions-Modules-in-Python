Task Name

Calculate Factorial Using a Function

Description
This program calculates the factorial of a given number using a user-defined function in Python.
The function takes a number as input, calculates its factorial using a loop, and returns the result.

Objectives
To understand the concept of functions in Python
To implement a function that returns a value
To use loops inside a function

Technologies Used
Python 3.x

How the Program Works
The user enters a number.

The program calls the factorial() function.
The function calculates the factorial using a for loop.
The result is returned and displayed.

Sample Input
5

Sample Output
Factorial of 5 is: 120

Conclusion
This program demonstrates the use of functions to perform repetitive mathematical operations efficiently and improves code reusability.
